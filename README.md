This is a banking management system project in java.
we can perform the following operations once the user is created.

Note :The Program automatically generates an Account Number and a PIN for every user.

A.Deposit Money.

B.Withdraw Money.

C.View User Details.

D.Settings option
-Forgot Pin.
-Change name.

E.Login.
